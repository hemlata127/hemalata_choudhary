3.
What will be the result of the following program?
main()
{
	int a,b;
	printf("Enter two numbers: ");
	scanf("%d%d",a,b);
	printf("%d+%d=%d",a,b,a+b);
}
Ans.
run time error - a and b will have some garbage value.
values entered will try to be read into a memory location represented by these garbage values
this will cause coredump or runtime error
