PROJECT NAME
Make my sub application using C programming language

PROJECT DESCRIPTION
This project accepts a different options from the user required to prepare a subway.
The options are used or combined appropriately to control flow of program.
This project demonstrates some of the programming concepts like use of:
control structures, symbolic constants, bitwise OR operator

Following information is accepted from the user
1. Choice of patty
User is presented with number of options for patty.
One option needs to be selected. This is like a radio button control on a GUI application.
Options must be assigned symbolic constants as 1, 2, 3, 4, 5, etc.
2. Choice of bread
User is presented with number of choices of bread.
One option needs to be selected. This is like a radio button control on a GUI application.
Options must be assigned symbolic constants as 1, 2, 3, 4, 5, etc.
3. Choice of bread preparation
User is presented with number of choices of bread preparation.
One option needs to be selected. This is like a radio button control on a GUI application.
Options must be assigned symbolic constants as 1, 2, 3, 4, 5, etc.
4. Choice of veggies
User is presented with a number of options for veggies.
Multiple options can be selected. This is like check box control on a GUI application
Options must be assigned symbolic constants as 1, 2, 4, 8, 16, 32, 61, 128, 256, etc.
Use OR (|) operator to combine the choices.
5. Choice of sauces
User can selected more than one options for sauces.
Multiple options can be selected. This is like check box control on a GUI application.
Use OR (|) operator to combine the choices.
Options must be assigned symbolic constants as 1, 2, 4, 8, 16, 32, 61, 128, 256, etc.
6. Choice of add-ons
User is presented with number of options for add-ons
Multiple options can be selected. This is like check box control on a GUI application.
Use OR (|) operator to combine the choices.
Options must be assigned symbolic constants as 1, 2, 4, 8, 16, 32, 61, 128, 256, etc.
