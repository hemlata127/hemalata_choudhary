Project Name: Customised Virtual File System
Application type: CUI application
Description:
	A virtual file system is created using preconfigured details like fixed number of inodes. An inode holds all the details of a file including file data. A super block structure instance holds details about the total number of inodes and total number of free inodes. Project provides help documentation about the commands supported. An infinite while loop is used to implement a shell which accepts different commands supported. The project supports following operations:
	1. Create file
	2. Remove file
	3. Read file
	4. Write file
	5. Open file
	6. Close file
	7. Close all files
	8. List files
	9. Stat file (fetch file statistics)
	10. Truncate file
